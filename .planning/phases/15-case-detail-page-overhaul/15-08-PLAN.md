**Activity Logging Pattern:**

Follow the same pattern as add-note-modal.tsx:

```typescript
// Primary approach: POST to activity endpoint
await apiClient.post(`/cases/${caseId}/activity`, {
  action: "interview_logged",
  description: `Interview with ${intervieweeName}`,
  actionDescription: `Scheduled interview with ${intervieweeName} (${intervieweeType})`,
  metadata: {
    intervieweeName,
    intervieweeType,
    scheduledAt,
    notes,
  },
});
```

If case has investigations and a dedicated interview API exists (POST /investigation-interviews), use that as primary with activity logging as fallback.

```typescript
interface LogInterviewFormData {

  scheduledAt: string; // ISO date
  notes?: string;
  investigationId?: string;
}
```
